Ripple Mobile Emulator

Demo

Ripple is a multi-platform mobile environment emulator that runs in a web browser and is custom-tailored to HTML5 mobile application testing. Ripple aims to reduce the challenges being faced by mobile developers caused by today's platform fragmentation in the marketplace.

Here is how you use Ripple:

  1. Using the Chrome desktop browser, install the Ripple extension
  2. Load the URL of the Web content you wish to profile and debug in the Chrome browser.
  3. Right click on the Web content and select Emulator → Enable.
  4. When prompted, click on the "WebWorks" button.
